Map<String, List<EventHandler>> combineEventProviders(List<EventProvider> providers)

Kombiniert die Events aus mehreren EventProvidern in einer Map

Source 

Map<String, List<EventHandler>> combineEventProviders(
    List<EventProvider> providers) {
  final Map<String, List<EventHandler>> combined = {};

  for (EventProvider p in providers) {
    combined.addAll(p());
  }

  return combined;
}
